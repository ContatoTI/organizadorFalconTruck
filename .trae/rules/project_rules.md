Neste projeto, utilize exclusivamente a biblioteca de componentes Shadcn UI para toda a construção de interface (UI).

Todos os elementos visuais (botões, inputs, cards, diálogos, tabelas, etc.) devem ser criados usando componentes do Shadcn UI.

Não use HTML ou CSS puro para componentes que já existam no Shadcn UI.

Respeite os temas, tokens, cores e tipografia fornecidos pelo ThemeProvider do Shadcn UI.

Reaproveite ao máximo os componentes e utilitários do Shadcn UI, seguindo sua documentação oficial.

Caso seja necessário criar um componente customizado, faça usando a base do Shadcn UI e mantendo a compatibilidade visual e de comportamento. Sempre respeitando e reutilizando as cores que já estão sendo utilizadas

Explique na resposta quando um recurso visual não estiver disponível na biblioteca, e sugira a alternativa mais próxima possível dentro do Shadcn UI.

Nunca misture múltiplas bibliotecas de UI sem justificativa e aprovação prévia.
