# Organizador - Next.js

Seu organizador pessoal de tarefas, finanças e metas.

## Setup

Instale as dependências:

```bash
npm install
```

## Development Server

Inicie o servidor de desenvolvimento em `http://localhost:4000`:

```bash
npm run dev
```

## Production

Crie a build de produção:

```bash
npm run build
```

Inicie o servidor de produção:

```bash
npm start
```
